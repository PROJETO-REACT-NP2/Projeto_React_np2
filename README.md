# 1️⃣ Contextualização

O projeto tem como objetivo o desenvolvimento de uma aplicação web frontend para:

* 🧠 Psicólogos
* ⚖️ Advogados
* 🏗️ Arquitetos

A aplicação compara a tributação entre:

* 👤 Pessoa Física (PF)
* 🏢 Pessoa Jurídica (PJ)

💡 Baseado em:

* 💰 Renda mensal
* 📉 Custos

---

# 2️⃣ Descrição do Problema

Facilitar o entendimento tributário e auxiliar profissionais na escolha do melhor regime.
